Colours

This page contains the default 22 colours from TailwindCSS plus over 60 additional colours (2023-11-05).

When using these colours the normal "BASE" or "DEFAULT" value will be the 500 point.

Some colours will have had the DEFAULT moved from the "500" midpoint to a value above or below. This is to provide the full 50-950 range of tints and shades.

There are also cases where the tints and shades have been adjusted to ensure that the "50" point is not white (#FFFFFF) and the "950" point is not black (#000000).

DEFAULT colours have also been provided by staff and students at NMTAFE, and added to this list using the following websites:

  • https://uicolors.app/create
  • https://www.tailwindshades.com/
  • https://coolors.co/colors
  • https://colorcodes.io/
  • https://www.flatuicolorpicker.com/
